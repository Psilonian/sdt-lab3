# SDT Lab 3 — расширение для VS Code

Это расширение для Visual Studio Code, созданное в рамках лабораторной работы по ИСРПО.  
Оно автоматически создаёт ветку `deadline_0` в текущем Git-репозитории и переключается на неё.
Расширение упрощает подготовку среды для выполнения лабораторных работ.
Использовался TypeScript;

---

## Возможности

- Создание ветки `deadline_0`, если она отсутствует.  
- Переключение на уже существующую ветку `deadline_0`.  
- Отображение уведомлений об успешных действиях или ошибках.  

Команда запускается через **Command Palette** → **“Create and checkout 'deadline_0'”**  
(горячие клавиши: **Ctrl + Shift + P** или **Cmd + Shift + P**).

Пример уведомления:

```Created and checked out ‘deadline_0’.```

---

## Требования

- Установленный **Git** и добавленный в переменную окружения `PATH`.  
- Открытая в VS Code папка должна быть инициализированным **Git-репозиторием** (должен быть хотя бы один коммит).  

---

## Настройки

На данный момент расширение не имеет настраиваемых параметров.  
Все действия выполняются для ветки `deadline_0`.

---

## Известные проблемы

- Если в репозитории нет коммитов, создание ветки невозможно.  
- При открытии нескольких рабочих папок используется только первая.
